--- layout: articles title: 코딩 테스트 articles: data_source: site.categories.코딩 테스트 show_cover: false show_excerpt: true show_readmore: true show_info: true sidebar: nav: study-nav --- 코딩테스트는 다음의 3단계를 차근차근 밟아갈 것. 1. 대표유형 익히기 2. 어떤 유형으로 풀지 감 잡기 3. 구현력 키우기 (어떤 유형으로 풀지 안 다음 그것을 구현해 내는 능력) 초기에는 실력을 빠르게 키우기 위해 문제를 끝까지 풀기보다는 풀이를 참고해가며 1, 2 단계를 빠르게 거치도록 하자.
