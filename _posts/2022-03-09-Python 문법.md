# Immutable, Mutable (중요)

![](/images/2022-03-09-16-14-50.png)

- b 변수의 값을 100으로 변경하여도!    메모리에 할당되었으며 99 라는 값을 가진 Immutable **객체**의 `값이 바뀌지 않음`
  - 단지 b라는 변수는 이제부터 새로운 주소의 99라는 값을 가진 Immutable **객체**를 참조함

- arr2 변수의 값을 변경하면!   메모리에 할당되었으며 [1,2,3] 이라는 값을 가진 Mutable **객체**의 `값이 변경됨`

- 아래 코드를 통해 정확히 이해해보자

```python
print("Immutable 객체")
a = 99
b = 99
c = 99
d = 99
e = 99

print(hex(id(a)))
print(hex(id(b)))
print(hex(id(c)))
print(hex(id(d)))
print(hex(id(e)))

a = 100

print(a, hex(id(a)))
print(b, hex(id(b)))
print(c, hex(id(c)))
print(d, hex(id(d)))
print(e, hex(id(e)))

print("Mutable 객체")

arr1 = [1, 2, 3]
arr2 = [1, 2, 3]
arr3 = [1, 2, 3]
arr4 = [1, 2, 3]

print(hex(id(arr1)))
print(hex(id(arr2)))
print(hex(id(arr3)))
print(hex(id(arr4)))
```
