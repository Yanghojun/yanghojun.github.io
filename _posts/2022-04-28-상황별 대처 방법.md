# 외국 Repository 주소 한국으로 변경

```bash
sudo vi /etc/apt/sources.list

:%s/archive.ubuntu.com/ftp.daum.net/g
:%s/security.ubuntu.com/ftp.daum.net/g 
:%s/extras.ubuntu.com/ftp.daum.net/g

sudo apt-get update 
sudo apt-get upgrade
```
