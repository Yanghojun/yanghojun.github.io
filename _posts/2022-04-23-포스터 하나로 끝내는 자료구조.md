# Queue
First In First Out (선입선출 구조)

<p align="center"> <img src="<p align="center"> <img src="../images/20220423160926.png" width="80%"> </p>" width="70%"> </p>

## 코드
- deque 라이브러리 사용하여 구현
  - deque로는 stack도 사실 구현 가능하다


```python
from collections import deque

queue = deque()

queue.append(5)
queue.append(2)
queue.append(3)
queue.append(7)
queue.popleft()
queue.append(1)
queue.append(4)
queue.popleft()

print(queue)
queue.reverse()
print(queue)
```

```
deque([3, 7, 1, 4])
deque([4, 1, 7, 3])
```

# 재귀 함수

자기 자신을 다시 호출하는 함수.  
내부적으로는 스택 자료구조 활용함.  
구현에 필수적으로 필요한 건 재귀적 호출이 끝나는 **종료조건**

<p align="center"> <img src="../images/20220423162302.png" width="70%"> </p>

위 그림은 재귀적으로 호출할 때 **인자값을 하나씩 올려서 넘겨주고**, 그 올라가는 인자값을 **종료조건**으로 활용하고 있는 것이다.

## 코드

def recursive_function(i):

recursive_function(0)

- 팩토리얼 구현을 통한 심화 이해

```python
def fact(n):
  num = 1
  for i in range(n):
    num*=(i+1)
  return num


def recur_fact(n):
  if n <= 1:
    return 1
  
  return n * recur_fact(n-1)
  
  
print(fact(5))
print(recur_fact(5))
```

```
120
120
```

# Graph(Node, Edge)

Graph를 표현하는 방식은 인접행렬(Adjacency Matrix) 방법, 인접 리스트(Adjacency List) 방법 존재

## Adjacency Matrix

2차원 배열로 그래프의 연결 관계를 표현하는 방식

||1|2|3|
|---|---|---|---|
||0|1|2|
|0|0|7|5|
|1|7|0|무한|
|2|5|무한|0|

### 코드

```python
INF = 999999999

# 2차원 리스트를 활용해 인접 행렬 표현
graph = [
    [0, 7, 5],
    [7, 0, INF],
    [5, INF, 0]
]

print(graph)
```

```
[[0, 7, 5], [7, 0, 999999999], [5, 999999999, 0]]
```

## Adjacency List

**모든 노드**에 연결된 노드에 대한 정보를 차례대로 연결하여 저장  

<p align="center"> <img src="../images/20220423204310.png" width="40%"> </p>

### 코드

```python
# 행(Row)이 3개인 2차원 리스트로 인접 리스트 ㅍ현
graph = [[] for _ in range(3)]

# 노드 0에 연결된 노드 정보 저장(노드, 거리)
graph[0].append((1, 7))
graph[0].append((2, 5))

# 노드 1에 연결된 노드 정보 저장(노드, 거리)
graph[1].append((0,7))

# 노드 2에 연결된 노드 정보 저장(노드, 거리)
graph[2].append((0, 5))

print(graph)
```

```
[[(1, 7), (2, 5)], [(0, 7)], [(0, 5)]]
```

파이썬 기본 문법으로 인접 행렬 방식, 인접 리스트 방식 모두 구현 가능
{:.info}

메모리 측면에서는 인접 리스트 방식의 이점이 크지만  
속도 측면에서는 인접 행렬 방식의 이점이 크다.
