# Collaborative filtering
가장 많이 사용되는 추천시스템 기법으로, 사용자 개인의 성향과 비슷한 성향을 가진 다른 사람들의 데이터를 활용해서 사용자에게 추천해주며 Memory based, Model based 방식으로 나뉨.

## Memory based 방식
Similarity를 이용함.  

## Model based 방식
SVD, KNN과 같이 이미 구현되어있는 Model 이용함.
