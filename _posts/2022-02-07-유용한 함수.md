# np.unique
- 어떤 값들이 데이터에 들어있는지 알 수 있음. 추가적으로 각 데이터의 갯수 또한 return_counts 인자로 알 수 있음
```python
unique, counts = np.unique(y, return_counts = True)
print(unique, counts)
```
